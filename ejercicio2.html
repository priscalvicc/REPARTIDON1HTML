HTML: lenguaje de marcado estándar para crear la estructura y el contenido de las páginas web

que es HTML: es el código estándar utilizado para estructurar y desplegar páginas web y sus contenidos, como textos, imágenes, videos y enlaces

historia de HTML: fue creado por Tim Berners-Lee en 1989-1991 en el CERN para facilitar el intercambio de documentos científicos. Evolucionó de un sistema simple a estándares complejos gestionados por el W3C desde 1994, destacando hitos como HTML 3.2 (1997), HTML 4.0 (1998) y el moderno HTML5, que introdujo mejoras multimedia y semánticas.

HTML 5: HTML5 es la quinta revisión del lenguaje de marcado de hipertexto (HTML), estándar para estructurar y presentar contenidos en la web
